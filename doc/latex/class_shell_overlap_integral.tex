\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_shell_overlap_integral}{Shell\+Overlap\+Integral}} class provides a way to calculate electron shell overlap integrals between two shells, represented by the \mbox{\hyperlink{class_shell}{Shell}} class.

\begin{DoxyNote}{Note}
The \mbox{\hyperlink{class_shell_overlap_integral}{Shell\+Overlap\+Integral}} class depends on the \mbox{\hyperlink{class_shell}{Shell}} class, which should be included in the \char`\"{}shell.\+h\char`\"{} header file.
\end{DoxyNote}
