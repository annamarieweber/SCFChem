◆ readfile()

Cluster util::io::readfile ( string &  filename)

reads molecule files where the first line is the number of molecules and the remaining lines follow the format below: atomic_number x_coord y_coord z_coord

atomic_numbers must be > 0 and < MAX_ATOMIC_NUM (Set by MAX_ATOMIC_NUM environment variable) atomic_numbers can be restricted with ALLOWED_ATOMIC_NUM environment variable which is a list of allowed atomic_numbers 

the number of atoms described by the file must match the specified number of atoms in the first line
Parameters
filename(string) the name of the file containing molecule data for the cluster
